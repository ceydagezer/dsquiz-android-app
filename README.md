 📚 Algorithms & Data Structures Quiz App (Android)

An interactive Android quiz application designed to help students practice **Algorithms and Data Structures** with categorized topics, difficulty levels, explanations, and progress tracking.

Built using **Kotlin + Jetpack Compose** 🚀


 Features:

 📊 Quiz Modes

- Data Structures category quiz
- Algorithms category quiz
- Subtopic-based quizzes:
  - Arrays
  - Linked Lists
  - Stacks
  - Queues
  - Trees
  - Graphs
  - Sorting
  - Searching
  - Recursion
- Mixed-topic quiz mode


 🎯 Difficulty Options

- Easy
- Medium
- Hard
- Mixed Difficulty Mode


 🧠 Learning Support

- Explanation after each question
- Retry Wrong Answers mode
- Mixed difficulty practice
- Performance feedback after quiz completion


 📈 Progress Tracking

- Score calculation
- Percentage result display
- Quiz history tracking
- Persistent storage using SharedPreferences
- Color-highlighted success levels in history screen


 🎨 User Experience

- Dark mode support 🌙
- Progress indicator during quiz
- Exit confirmation dialog
- Clean Material 3 UI
- Smooth navigation between screens


 🧩 Question Bank

Includes structured questions from:

- Arrays
- Linked Lists
- Stacks
- Queues
- Trees
- Graphs
- Sorting Algorithms
- Searching Algorithms
- Recursion

📊 Total questions: 90


 🛠️ Technologies Used

- Kotlin
- Jetpack Compose
- Navigation Component
- Material 3
- SharedPreferences
- Compose State Management
